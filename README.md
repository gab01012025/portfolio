# Portfolio - Gabriel Barreto

Portfolio pessoal desenvolvido com Next.js 15, TypeScript e Tailwind CSS.

## Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Framer Motion
- EmailJS (formulário de contato)

## Rodar localmente

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`

## Deploy

Deploy automático na Vercel via GitHub.
